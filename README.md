# JavaScript Array Methods II 🚀

This repository contains practical examples and exercises for mastering essential JavaScript ES6+ features and array methods. It’s designed to help understand how to handle dynamic data efficiently.

## 📝 Topics Covered

### 1. Rest & Spread Operators
- **Rest Operator (`...`):** Used in function parameters to gather multiple arguments into a single array (e.g., `generateBill` function).
- **Spread Operator (`...`):** Used to unpack elements from an array or combine multiple arrays into one.

### 2. Map Method
- Transforming data: Example includes converting USD to INR.
- **Task Included:** Logic for adding GST to price lists.

### 3. Filter Method
- Selecting specific data based on conditions.
- Examples include filtering a food menu based on:
  - Price (e.g., items under ₹300)
  - Category (Veg/Non-Veg)
  - Specific item names

### 4. Reduce Method
- Used for accumulator logic.
- **Cart Logic:** Calculating the total bill amount from an array of prices.
- **Algorithm:** Combining two different numeric arrays and finding their grand total.

## 🛠️ How to Use
1. Clone the repository.
2. Open `index.html` in your browser.
3. Open the **Developer Console (F12)** to see the output of each method.

## 🎯 Purpose
To practice functional programming concepts in JavaScript and understand how to manipulate arrays like a pro!
